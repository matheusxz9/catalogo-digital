## 📌 About the Project

**Digital Catalog** is a platform developed to simplify the visualization and organization of products in a modern and intuitive way.

The project was built with a strong focus on responsiveness, performance, and user experience, providing smooth navigation across different devices.

The application features a clean and professional interface, making it ideal for online stores, digital showcases, and product presentations.

---

## ✨ Features

* 📦 Dynamic product display
* 🔍 Search and filtering system
* 📱 Fully responsive interface
* ⚡ High performance and fast loading
* 🎨 Modern and professional design
* 🖥️ Optimized layout for desktop and mobile
* 🚀 Scalable structure for future features

## ⚙️ Installation and Setup

Clone the repository:

```bash
git clone https://github.com/matheusxz9/catalogo-digital.git
```

Access the project folder:

```bash
cd catalogo-digital
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
